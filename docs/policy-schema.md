# Policy Schema

Required fields:

- `name`: policy name
- `version`: semantic or internal version string

Common optional fields:

- `allowed_tools`
- `blocked_tool_names`
- `tools_requiring_approval`
- `max_autonomous_amounts`
- `tool_role_requirements`
- `approval_tool_environments`
- `blocked_tool_environments`
- `approval_routes`
- `approval_expiration_minutes`
- `blocked_content_patterns`
- `completion_requires_artifacts`
- `factual_claims_require_sources`
